# 如何运行节点
## 运行Publisher
1. 打开终端1进入到工作空间  
`cd ros2_homework`
2. 刷新环境变量  
`. install/setup.bash`
3. 运行publisher节点  
`ros2 run py_homework02 homework02_talker_py`
## 运行Listener
1. 打开终端2进入到工作空间  
`cd ros2_homework`
2. 刷新环境变量  
`. install/setup.bash`
3. 运行Listener节点  
`ros2 run py_homework02 homework02_listener_py`
